# TimeSeries HW3

Влияние нормализации на качество прогнозирования временных рядов: сравнение локальных и глобальных моделей
## Обоснование пайплайна

Экспериментальный процесс выстроен вокруг задачи сравнения предсказательной способности моделей при использовании четырех стратегий подготовки данных. В качестве контрольной точки зафиксированы результаты работы на сырых данных без применения трансформаций. Основной этап включает интеграцию StandardScaling для оценки базовой необходимости приведения к единому масштабу. Для анализа устойчивости моделей к выбросам и форме распределения в пайплайн включены методы RobustScaling и QuantileScaling.

Пайплайн реализует дифференцированный подход к оценке моделей разных классов. Это позволяет выявить специфические зависимости. Например, критическую важность нормализации для оптимизации весов нейронных сетей и потенциальную избыточность или вред трансформаций для решающих деревьев в условиях глобального обучения. Математическая корректность обеспечивается за счет вычисления параметров скейлинга исключительно на обучающей выборке и обязательного Inverse Transform перед расчетом метрик.

| Модуль | Назначение | Техническая реализация |
| :--- | :--- | :--- |
| **run_experiment.py** | Оркестрация | Управление потоками данных и сохранение артефактов |
| **analysis_results.py** | Аналитика | Генерация векторной графики на базе сохраненных весов и прогнозов |
| **src/preprocessing.py** | Трансформация | Реализация TimeSeriesScaler с поддержкой Inverse Transform |

| **src/evaluate.py** | Оценка | Векторизованный расчет деградации ошибки по горизонту прогноза |
| **config.py** | Конфигурация | Хранение гиперпараметров и настроек обучающей выборки |

## Reference

Проект опирается на современные исследования в области трансформерных архитектур и градиентного бустинга.

* **PatchTST:** Nie et al. 2023. A Time Series is Worth 64 Words. https://arxiv.org/abs/2211.14730
* **CatBoost:** Prokhorenkova et al. 2018. CatBoost: unbiased boosting with categorical features. https://arxiv.org/abs/1706.09516
* **M4 Dataset:** Makridakis et al. 2018. The M4 Competition: Results findings conclusion and way forward. https://doi.org/10.1016/j.ijforecast.2018.06.001

> TL;DR: Экспериментально доказано что использование StandardScaler для глобального CatBoost снижает sMAPE на 65 процентов. Глобальные модели требуют унификации масштаба для эффективного поиска порогов в узлах деревьев. Модель PatchTST демонстрирует стабильность при использовании RobustScaler на зашумленных данных но требует больших объемов выборки для превосходства над сезонными бейзлайнами.
